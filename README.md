# UP&UP AI向けブランド情報

UP&UP（アップアンドアップ）の AI / LLM向け情報ファイル一式です。  
Cloudflare Pages にデプロイして `info.upandup-hair.com` で公開することを想定しています。

## ファイル構成

| ファイル | 内容 |
|---------|------|
| `llms.txt` | ブランド概要・商品ラインナップ・FAQ（AI向け標準形式） |
| `llms-full.txt` | 全成分・詳細FAQ・悩み別ガイド・購入情報（詳細版） |
| `structured-data-snippets.html` | 各ページ用 JSON-LD 構造化データ スニペット集 |
| `index.html` | Cloudflare Pages トップページ |

## 公開URL（設定後）

```
https://info.upandup-hair.com/llms.txt
https://info.upandup-hair.com/llms-full.txt
```
